## Setup Guide
1. Create a new Unity project (I recommend Unity 6000.0.43f1 or later)
2. Open the Package Manager window (Window > Package Manager)
3. Click on "My Assets"
4. Download & Install Mythril2D

## Documentation
You can read more about Mythril2D and learn to create your own games with it by reading the editor built-in documentation, accessible from Unity's menu bar: `Window > Mythril2D > Documentation`.
