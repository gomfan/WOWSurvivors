# Conditional Activators
> On this page, we'll learn about Mythril2D conditional activators.

**ℹ️ If you aren't familiar with them already, I'd recommend reading about "Conditions" before looking into conditional activators!**

## ConditionalChildrenActivator <`AConditionalActivator`>
Enable or disable all its children if a condition is met.

## ConditionalReferencesActivator <`AConditionalActivator`>
Enable or disable all its attached references if a condition is met.
